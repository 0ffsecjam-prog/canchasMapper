Volver al mapa

API keys

Las keys se guardan en la base local (SQLite) y se usan tanto para syncs como para el render del mapa cuando elijas Google Maps. Podés generarlas en Google Cloud Console.

Fuentes de datos

OSM (Overpass) es gratis y no requiere key. Google Places requiere el server key configurado arriba. Los syncs corren en background; el avance se muestra debajo.

Remapear todo

Corre zonas + OSM + Google (si hay key) + reasignación, en orden, de una sola vez.

OpenStreetMap

Cubre AMBA completo. Tarda 1-3 minutos.

Google Places

Más fotos y datos, pero gasta cuota. Tarda varios minutos.

Zonas (límites)

Barrios CABA + partidos GBA. Necesario para filtrar y dibujar contornos.

Exportar todo

Descarga el dataset completo (sin filtros). Para exportar con filtros específicos usá los botones del mapa en la home.
